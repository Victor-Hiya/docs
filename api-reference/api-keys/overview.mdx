API keys belong to [users][2] and provide long-lived [bearer tokens][1],
which in turn can be used to authenticate API requests.

<Tip>
  To learn how to use the token of an API key to
  authenticate your API requests, read the
  [bearer tokens][1] page.
</Tip>

<Note>**Access:** Only you can manage your API keys.</Note>
